Features

Haltbarkeit Überwachung

Das System verlässt sich nicht blind auf das aufgedruckte Datum, sondern berechnet die Haltbarkeit dynamisch anhand des Zustands und des Lagerorts.

Mecker Dashboard

Das Dashboard holt sich die Daten von der Datenbank nach Login und auch sortiert zurück, welche Lebensmittel am ablaufen sind. Ein Ampel zeigt wie das Ausgewählte lager mit Lebensmitteln noch gut genießbare Produkte hat.

Soll-Ist Abgleich

Das System berechnet kontinuierlich die Differenz zwischen dem Soll- und Ist-Bestand der Lebensmittel. Hierdurch wird der Benutzer frühzeitig und automatisiert darüber informiert, welche Vorräte zur Neige gehen, um Engpässe zu vermeiden.

Kontoerstellung

Die Erstellung eines Benutzerkontos ermöglicht es, die erfassten Daten dauerhaft in einer zentralen Datenbank zu speichern. Hierdurch wird eine plattformunabhängige Synchronisation gewährleistet, sodass der Benutzer von überall auf seine Bestände zugreifen kann.

Nachhaltige Vermeidung von Lebensmittelverschwendung

Durch den gezielten Einsatz unserer Anwendung wird die Verschwendung von Lebensmitteln im Haushalt nachweislich minimiert. Das System unterstützt den Nutzer aktiv dabei, Produkte rechtzeitig zu verbrauchen und Abfälle zu reduzieren.

Preisleistung

Durch ein kostengünstiges Nutzungsmodell wird die kontinuierliche Weiterentwicklung der Anwendung sichergestellt. Gleichzeitig leistet jedes Abonnement einen aktiven Beitrag zur nachhaltigen Reduzierung von Lebensmittelverschwendung.
